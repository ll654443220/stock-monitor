# 盯盘助手 - 板块资金流向实时监控

纯前端 PWA 应用，无需后端，数据来自新浪财经/腾讯财经公开 API。

## 功能

- 📊 大盘指数实时行情（上证、深证、创业板、科创50、沪深300、国证2000）
- 🏭 49个行业板块行情，支持按涨跌幅/成交额排序
- 💰 主力资金流向排行
- 🔥 今日涨幅榜 TOP 20
- 🔍 股票搜索
- 📱 PWA 支持，可添加到 Android/iOS 桌面
- 🌙 暗色主题，交易时段自动刷新

## 使用

直接打开 `index.html` 即可，或部署到 GitHub Pages。

## Android 安装

1. 用 Chrome 打开页面
2. 点击菜单 → **添加到主屏幕**
3. 桌面图标打开即全屏体验

## 数据来源

- 新浪财经（板块行情、资金流向、搜索）
- 腾讯财经（个股行情）
